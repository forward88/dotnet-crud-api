
# Show Categories

Returns a list of Categories.

**URL** : `/api/v{:apiVersion}/categories`

**Method** : `GET`

**Authorization** : `Bearer Token`


## Success Response

**Code** : `200 OK`

**Content Example** :

```json
[
    {
        "id": 1,
        "name": "Phone"
    },
    {
        "id": 2,
        "name": "Computer"
    },
    {
        "id": 3,
        "name": "Home & Kitchen"
    }
]
```
